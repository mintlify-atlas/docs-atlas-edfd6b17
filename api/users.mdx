## Get User Profile

<CodeGroup>
```bash cURL
curl "http://localhost:8000/users/user_1"
```

```python Python
import requests

response = requests.get("http://localhost:8000/users/user_1")
print(response.json())
```
</CodeGroup>

### Path Parameters

<ParamField path="user_id" type="string" required>
  The unique identifier of the user to retrieve
</ParamField>

### Response

<ResponseField name="user" type="object">
  User profile information
  
  <Expandable title="User Object">
    <ResponseField name="id" type="string">
      User identifier
    </ResponseField>
    
    <ResponseField name="name" type="string">
      User's full name
    </ResponseField>
    
    <ResponseField name="avatar" type="string">
      User's avatar image URL
    </ResponseField>
    
    <ResponseField name="bio" type="string">
      User biography/description
    </ResponseField>
    
    <ResponseField name="interests" type="array">
      Array of user interest categories (e.g., ["Cafe", "Restaurant", "Art Gallery"])
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="interested_venues" type="array">
  List of venues the user is interested in
  
  <Expandable title="Venue Object">
    <ResponseField name="id" type="string">
      Venue identifier
    </ResponseField>
    
    <ResponseField name="name" type="string">
      Venue name
    </ResponseField>
    
    <ResponseField name="category" type="string">
      Venue category
    </ResponseField>
    
    <ResponseField name="description" type="string">
      Venue description
    </ResponseField>
    
    <ResponseField name="image" type="string">
      Primary image URL
    </ResponseField>
    
    <ResponseField name="images" type="array">
      Array of image URLs for galleries
    </ResponseField>
    
    <ResponseField name="address" type="string">
      Venue address
    </ResponseField>
    
    <ResponseField name="latitude" type="float">
      Latitude coordinate
    </ResponseField>
    
    <ResponseField name="longitude" type="float">
      Longitude coordinate
    </ResponseField>
    
    <ResponseField name="interested_count" type="integer">
      Total users interested
    </ResponseField>
    
    <ResponseField name="distance_km" type="float">
      Distance from user in kilometers
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="action_items" type="array">
  Active action items for venues the user is interested in
  
  <Expandable title="Action Item Object">
    <ResponseField name="id" type="string">
      Action item identifier
    </ResponseField>
    
    <ResponseField name="venue_id" type="string">
      Associated venue ID
    </ResponseField>
    
    <ResponseField name="interested_user_ids" type="array">
      Array of user IDs interested in this venue
    </ResponseField>
    
    <ResponseField name="action_type" type="string">
      Type of action
    </ResponseField>
    
    <ResponseField name="action_code" type="string">
      Action code (e.g., "GROUP_VISIT")
    </ResponseField>
    
    <ResponseField name="description" type="string">
      Human-readable action description
    </ResponseField>
    
    <ResponseField name="threshold_met" type="boolean">
      Whether the threshold was met
    </ResponseField>
    
    <ResponseField name="status" type="string">
      Status ("active", "completed", "dismissed", "expired")
    </ResponseField>
    
    <ResponseField name="created_at" type="string">
      ISO timestamp of creation
    </ResponseField>
    
    <ResponseField name="venue" type="object">
      Venue details associated with the action item
    </ResponseField>
  </Expandable>
</ResponseField>

## Get User Action Items

<CodeGroup>
```bash cURL
curl "http://localhost:8000/users/user_1/action-items"
```

```python Python
import requests

response = requests.get("http://localhost:8000/users/user_1/action-items")
print(response.json())
```
</CodeGroup>

### Path Parameters

<ParamField path="user_id" type="string" required>
  The unique identifier of the user
</ParamField>

### Response

<ResponseField name="action_items" type="array">
  Array of active action items where the user is in interested_user_ids
</ResponseField>

<ResponseField name="count" type="integer">
  Number of active action items
</ResponseField>

```python Python
import requests
```

</CodeGroup>

<ParamField path="user_id" type="string" required>

## Get User Chats


```python Python
import requests

response = requests.get("http://localhost:8000/users/user_1/chats")
```

</CodeGroup>

### Path Parameters

<ParamField path="user_id" type="string" required>
  The unique identifier of the user
</ParamField>

### Response

  Array of chat objects where the user is a participant
