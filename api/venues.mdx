## List All Venues

<CodeGroup>
```bash cURL
curl "http://localhost:8000/venues?user_id=user_1&categories=Cafe&max_distance=5&sort_by=distance"
```

```python Python
import requests

response = requests.get(
    "http://localhost:8000/venues",
    params={
        "user_id": "user_1",
        "categories": "Cafe,Restaurant",
        "max_distance": 5,
        "sort_by": "distance"
    }
)
print(response.json())
```
</CodeGroup>

### Query Parameters

<ParamField query="user_id" type="string" optional>
  User ID to calculate distances and friend interests. When provided, enables distance-based filtering and sorting.
</ParamField>

<ParamField query="categories" type="string" optional>
  Comma-separated list of categories to filter by (e.g., "Cafe,Restaurant"). Use "all" to show all categories.
</ParamField>

<ParamField query="max_distance" type="float" optional>
  Maximum distance in kilometers from user location. Requires `user_id` to be provided.
</ParamField>

<ParamField query="min_friend_interest" type="integer" optional>
  Minimum number of friends interested in the venue. Requires `user_id` to be provided.
</ParamField>

<ParamField query="only_interested" type="boolean" optional>
  Show only venues the user is interested in. Requires `user_id` to be provided.
</ParamField>

<ParamField query="exclude_interested" type="boolean" optional>
  Show only venues the user is NOT interested in. Requires `user_id` to be provided.
</ParamField>

<ParamField query="sort_by" type="string" optional default="popularity">
  Sort criterion:
  - `popularity` - Sort by number of interested users (default)
  - `distance` - Sort by distance from user (requires `user_id`)
  - `friends` - Sort by number of friends interested (requires `user_id`)
  - `name` - Sort alphabetically by name
</ParamField>

### Response

<ResponseField name="venues" type="array">
  Array of venue objects matching the filters
  
  <Expandable title="Venue Object">
    <ResponseField name="id" type="string">
      Unique venue identifier
    </ResponseField>
    
    <ResponseField name="name" type="string">
      Venue name
    </ResponseField>
    
    <ResponseField name="category" type="string">
      Venue category (e.g., "Cafe", "Restaurant", "Bar")
    </ResponseField>
    
    <ResponseField name="image" type="string">
      Primary image URL
    </ResponseField>
    
    <ResponseField name="interested_count" type="integer">
      Total number of users interested in this venue
    </ResponseField>
    
    <ResponseField name="created_at" type="string">
      ISO timestamp of when venue was created
    </ResponseField>
    
    <ResponseField name="distance_km" type="float" optional>
      Distance in kilometers from user location (only when `user_id` provided)
    </ResponseField>
    
    <ResponseField name="friends_interested" type="integer" optional>
      Number of user's friends interested (only when `user_id` provided)
    </ResponseField>
    
    <ResponseField name="user_interested" type="boolean" optional>
      Whether the user is interested in this venue (only when `user_id` provided)
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="applied_filters" type="object">
  Summary of filters applied to the query
</ResponseField>

<ResponseField name="result_count" type="integer">
  Number of venues returned after filtering
</ResponseField>

<ResponseField name="total_venues" type="integer">
  Total number of venues in the database
</ResponseField>

## Get Venue Details

<CodeGroup>
```bash cURL
curl "http://localhost:8000/venues/venue_1?user_id=user_1"
```

```python Python
import requests

response = requests.get(
    "http://localhost:8000/venues/venue_1",
    params={"user_id": "user_1"}
)
print(response.json())
```
</CodeGroup>

### Path Parameters

<ParamField path="venue_id" type="string" required>
  The unique identifier of the venue to retrieve
</ParamField>

### Query Parameters

<ParamField query="user_id" type="string" optional>
  User ID to calculate distance from user location
</ParamField>

### Response

<ResponseField name="venue" type="object">
  Complete venue details
  
  <Expandable title="Venue Details">
    <ResponseField name="id" type="string">
      Unique venue identifier
    </ResponseField>
    
    <ResponseField name="name" type="string">
      Venue name
    </ResponseField>
    
    <ResponseField name="category" type="string">
      Venue category
    </ResponseField>
    
    <ResponseField name="description" type="string">
      Full venue description
    </ResponseField>
    
    <ResponseField name="image" type="string">
      Primary image URL
    </ResponseField>
    
    <ResponseField name="images" type="array">
      Array of image URLs for multi-image galleries
    </ResponseField>
    
    <ResponseField name="address" type="string">
      Full venue address
    </ResponseField>
    
    <ResponseField name="latitude" type="float">
      Venue latitude coordinate (for "Get Directions" feature)
    </ResponseField>
    
    <ResponseField name="longitude" type="float">
      Venue longitude coordinate (for "Get Directions" feature)
    </ResponseField>
    
    <ResponseField name="distance_km" type="float" optional>
      Distance from user location in kilometers (only when `user_id` provided)
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="interested_users" type="array">
  List of users interested in this venue
  
  <Expandable title="User Object">
    <ResponseField name="id" type="string">
      User identifier
    </ResponseField>
    
    <ResponseField name="name" type="string">
      User name
    </ResponseField>
    
    <ResponseField name="avatar" type="string">
      User avatar URL
    </ResponseField>
  </Expandable>
</ResponseField>
