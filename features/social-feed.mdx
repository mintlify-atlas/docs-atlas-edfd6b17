The social activity feed enables you to discover what your friends are interested in and track venue popularity across your social network. Social features are integrated throughout the app, with the **Action Items** tab serving as the primary hub for social coordination.

<Note>
  **Location**: Social activity features are accessed primarily through the **Action Items** tab (third tab in main navigation), not a dedicated "Social" tab.
</Note>

## Hot Right Now

The "Hot Right Now" section highlights venues that have reached critical mass within your social network or across the platform:

### Threshold Requirements

<Card title="5+ Users Interested" icon="fire">
When 5 or more users (friends or platform-wide) express interest in the same venue, it triggers action items for coordination. These appear in the Action Items tab with priority status.
</Card>

### Where to Find Hot Venues

**Action Items Tab**:
- Venues with 5+ interested users automatically generate action items
- Action item cards show interested friend avatars
- Tap "Go Ahead" to confirm attendance and join coordination

**Profile Tab → Active Chats**:
- After confirming interest, group chats display venue coordination discussions
- Recent activity shows who's planning to attend

<Note>
The Hot Right Now concept focuses on action items rather than a dedicated feed section, streamlining the coordination experience.
</Note>

## Friend Activity Tracking

While Luna doesn't have a dedicated "Social" tab, friend activity is tracked through several mechanisms:

### Interest Activities

The backend `/social/feed` and `/activities` endpoints track when friends:
- Express interest in venues
- Confirm attendance via action items
- Join group chats for coordination

### Activity Visibility

Friend activities are surfaced in:

1. **Action Items Tab**: See which friends are interested in the same venues as you
2. **Venue Detail Pages**: "Interested Users" section shows friends who liked the venue
3. **Profile → Interested Venues**: Track your own activity history
4. **Group Chats**: Real-time coordination with confirmed friends

<Info>
  **API Available**: The `/social/feed` and `/activities` API endpoints are implemented and can be used for custom integrations or future UI enhancements.
</Info>

## Activity Timeline (Backend)

The backend provides activity feed endpoints even though a dedicated timeline UI isn't currently implemented:

### GET /social/feed

Returns structured social feed data including:
- **Interest activities** - Friend venue interests with timestamps
- **Highlighted venues** - Venues with 5+ interested friends
- Pagination support for scalable history
- Real-time updates via polling

### GET /activities

Provides raw activity stream:
- User interest events
- Chronological ordering
- Filterable by user or venue
- Supports infinite scroll pagination

<Tip>
  Developers can leverage these endpoints to build custom social feed visualizations or integrate activity data into other views.
</Tip>

## Auto-Refresh and Real-Time Updates

Social data updates automatically through various mechanisms:

### Action Items Polling

The Action Items tab:
- Auto-refreshes on pull-to-refresh
- Updates when returning from venue details
- Shows real-time interest count changes
- Badges indicate new items

### Chat Polling

Group chats poll every 5 seconds for:
- New messages from participants
- Updated participant lists
- Real-time conversation flow

### Interest Synchronization

When you or friends express interest:
1. Local UI updates optimistically (instant feedback)
2. Backend validates and persists interest
3. Action items trigger if threshold met
4. All interested users notified via their action items list

<Note>
  **Battery Optimization**: Polling only occurs when relevant views are active. Background updates are paused to conserve battery.
</Note>

## Social Discovery Patterns

Even without a dedicated social tab, Luna promotes social discovery through:

### Friend Interest Signals

**Recommendation Algorithm** (25% weight):
- Friend interest count influences venue scores
- Venues your friends like rank higher in "For You"
- Social signals make recommendations more relevant

**Discover Tab Sorting**:
- Sort by "Friend Count" to see popular venues among your network
- Filter by "Friends Interested" to show only socially validated venues

### Action Item Coordination

**Critical Mass Detection**:
- 5+ interested users triggers action items
- All interested parties notified simultaneously
- Streamlines group planning without manual coordination

**Confirmation Flow**:
- "Go Ahead" button confirms attendance
- 2+ confirmations auto-create group chat
- Real-time coordination within the app

## API Integration

For developers building on Luna's social features:

### Available Endpoints

- `GET /social/feed` - Comprehensive feed with highlights and activities
- `GET /activities` - Raw activity stream
- `POST /interests` - Creates activity when toggling interest
- `GET /users/{id}` - User profile with social graph

### Example: Fetch Social Feed

```swift
// Swift example using APIService
let feed = try await apiService.fetchSocialFeed(userId: currentUserId)

for activity in feed.interest_activities {
    print("\(activity.user.name) interested in \(activity.venue.name)")
}

for venue in feed.highlighted_venues {
    print("Hot: \(venue.name) with \(venue.interested_count) friends")
}
```

```python
# Python example using requests
import requests

response = requests.get(
    "http://localhost:8000/social/feed",
    params={"user_id": "user_1"}
)

feed = response.json()
print(f"Found {len(feed['interest_activities'])} activities")
```
