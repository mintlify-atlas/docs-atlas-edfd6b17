## Friend Network

All 15 demo users are friends with each other, creating a complete social graph. This means:
- Every user sees activities from 14 friends in their social feed
- Friend interest signals appear on all venue cards
- Action items can trigger when 5+ users from the network express interest

**Total Friendships**: 105 bidirectional connections (15 users × 14 friends each ÷ 2)

## Switching Between Users

You can switch between demo users at any time to test different perspectives:

<Steps>
  <Step title="Navigate to Profile Tab">
    Tap the **Profile** icon in the bottom tab bar (rightmost tab).
  </Step>
  
  <Step title="Tap User Name">
    Tap your current user name at the top of the profile screen.
  </Step>
  
  <Step title="Select New User">
    Choose any of the 15 demo users from the user switcher modal.
  </Step>
</Steps>

The app will immediately reload all data with the new user's perspective:
- Personalized recommendations based on their interests
- Distance calculations from their location
- Friend activity feed from their social network
- Action items relevant to their interested venues

## Data Persistence

<Warning>
**Local State Only**: Demo user data is stored in the backend's SQLite database (`luna.db`). Each time you switch users, the app fetches fresh data from the API.

To reset all user data:
1. Stop the backend server
2. Delete `Luna-Backend/luna.db`
3. Restart the server (database will be re-seeded automatically)
</Warning>
